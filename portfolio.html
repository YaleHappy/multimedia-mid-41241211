天氣預報API OpenData
HTML CSS JavaScript API

這個專案利用政府開放資料(OpenData)的天氣預報API，開發了一個可視化的天氣預報網頁應用。可查詢全台各地的即時天氣資訊和未來預報。

開發時間: 2024年2月

功能特點:

  • 整合中央氣象署OpenData API
  • 互動式地圖選擇地區
  • 即時天氣與未來預報顯示
  • 溫度、濕度和降雨機率等詳細資訊
